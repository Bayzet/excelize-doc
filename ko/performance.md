## 유사한 라이브러리의 성능 비교

다음 그래프는 Go, Python, Java, PHP 와 NodeJS 를 포함한 개인용 컴퓨터의 주요 오픈 소스 Excel libs 에 의한 세대 `102400*50` 일반 텍스트 매트릭스의 성능 비교를 보여줍니다 (2.6 GHz 6-Core Intel Core i7, 16 GB 2667 MHz DDR4, 500GB SSD, macOS Monterey 12.3.1).
